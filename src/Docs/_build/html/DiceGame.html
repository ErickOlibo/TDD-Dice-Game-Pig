class DiceGame.event.Event(name)[source]¶

Bases: object

Represents an event when a player has the hand in the game. The event is Unix Time stamped when the hand starts for every roll. A total point is calculated till the end of roll [Hold] or a roll of 1.

Args: name (str): The player’s name.

Attributes: _time_stamp (float): The Unix timestamp of the event. _name (str): The player’s name. _points (int): The total points accumulated during the event. _rolls (list): A list of the rolls made during the event.
